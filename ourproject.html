Prediction of Admission and Jobs in Engineering, Management and Pharmacy
with respect to Demographic Location

THE PROJECT

The target we are pursuing here, is Prediction of admission and jobs in demographic details on basis of some parameters. Our approach to the problem is through the supply and demand management using which we optimise the use of resources which are students, college seats and most importantly, jobs and minimise the cost function which is proportional to the number of vacant seats in the colleges. We have implemented this logic using a four layered feed forward neural network. For our web page we have used HTML and CSS for the front end and PHP in background for connecting our web pages with the web server. We have implemented a web based portal which can be used for two purposes- predicting jobs in a particular demographic location and also for seeing the availability of jobs in the desired location
